29 ## Canopen program
30 The executable file canopen enable you to send SDO message to a CAN bus.
31 ### usage:
32  ./canopen ifname 0xindex 0xsub [ size base data ]
33 
34  Arg: ifname : CAN interface name
35  0xindex : Object register index
36  0xsub : Object register subindex
37  size : Data size (number of bytes)
38  base : Numerical base of the value passed.
39  data : Value to write.
40 
41  Ex: To read register 0x1000:2 of node 4 on "can0":
42  ./canopen can0 4 1000 2
43 
44  To write in register 0x2000:F of node 3 the value 0x1234 on "can0":
45  ./canopen can0 3 2000 F 2 x 1234
